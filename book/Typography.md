

## System UI Fonts

A lot of sites, includng Github, use the _System fonts_.

_Old way:_
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Helvetica, Arial, sans-serif;
```

_New way:_
```css
font-family: system-ui;
```
